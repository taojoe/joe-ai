# Math Print Quiz Skill 指南

这个 Skill 用于帮助用户生成定制化的、A4 纸打印的数学知识讲解与小测验 PDF 文件。你需要作为一个教育辅助助手，引导用户生成高质量的内容。

## 🎯 工作流程 (Workflow)

请严格按照以下步骤执行此 Skill：

### Step 1: 需求讨论与主题确认
- **主动与用户沟通**：询问用户希望生成什么数学知识点的数据（例如：平方差公式、二次根式、一元二次方程等）。
- 确认要测试的范围以及是否需要特定类型的题目（如画图题）。该工具的主要目的是帮助学生**建立和完善数学概念**，并进行**基础练习**。
- **等待用户回复**，不要在没有明确主题的情况下生成数据。

### Step 2: 构思与生成 JSON 数据
在确认知识点后，你需要帮用户构思内容并生成对应的 JSON 配置文件。**重要：必须使用带有时间戳或自增数字作为前缀的文件名**（存放在 `data/<日期时间或数字前缀>_<主题英文名>.json`），以确保即使相同的知识点多次生成，也不会覆盖旧文件。例如：`data/20260331_1430_sum-of-cubes.json` 或 `data/01_difference-of-squares.json`。
生成的数据必须严格遵守以下**内容生成标准**：

1. **整体篇幅约定**：输出必须适配 A4 纸排版，共计分为 **4 页**（1 页知识，2 页练习，1 页答案）。
2. **第一部分（第 1 页）：知识讲解**
   - 详细讲解该知识点的原理，**必须从多个维度（或多种推导方法）展开**。例如：讲解立方和公式时，不能只给出一个干巴巴的推导；既要展示正向展开验证法（利用乘法分配律将右侧展开），也要引导代数推导法（由 $(a+b)^3$ 变形提取公因式推导），甚至可以提及其几何意义或实际应用。
   - 必须通过这种多角度的剖析，建立更扎实的心智模型，从而彻底讲透该知识点的来龙去脉。
   - 内容须极其充实，**必须完全撑起一整页的 A4 版面**，包含原理描述、多维推导过程、核心公式、避坑指南以及经典例题演示。
3. **第二部分（第 2、3 页）：练习题**
   - **题目数量要求**：生成 **4～6 道** 题目，这些题目会被平均分配到 2 页 A4 纸中，以确保有**非常大且充足的留白供孩子作答、演算或画图**（尤其是问答题，作答区域一定要大）。
   - **题目形式要求**：可以是选择题 (multiple-choice)、简答/计算题 (short-answer)，也可以是画图题 (drawing)（例如：引导孩子在坐标系或留白中画出函数的图像）。题目说明必须非常清晰。
4. **第三部分（第 4 页）：答案与解析**
   - 为每道题提供详尽的答案和解题过程。
   
**🚨 重要！LaTeX 转义规则**：
因为你要输出的是原生 JSON 文件，JSON 字符串将反斜杠作为转义符。因此你**必须将 LaTeX 公式中的单个反斜杠，在 JSON 中写成两个反斜杠 (\\)**。例如，必须写成 `\\times` 而不是 `\times`（否则会被解析为 Tab 制表符！）；写成 `\\frac` 而不是 `\frac`。但是请注意：**绝对不要写成四个反斜杠 `\\\\`**，否则会变成换行符！

### Step 3: 运行生成命令
当你完成了 JSON 文件的生成或更新后，你需要告诉用户即将运行脚本生成 PDF，并执行以下命令（注意文件名包含前缀）：
```bash
bun run quiz -- <带前缀的json文件名(不含扩展名)>
# 例: 如果生成了 data/20260331_1430_difference-of-squares.json
# 则运行: bun run quiz -- 20260331_1430_difference-of-squares
```

## 📄 数据格式参考 (`data/*.json`)

你在 Step 2 中生成的文件必须符合以下结构：

```json
{
  "title": "知识点中文标题",
  "subtitle": "英文副标题 (可选)",
  "knowledge": {
    "description": "知识说明 (支持 Markdown 与 LaTeX: $...$)",
    "formulas": ["核心公式1 (LaTeX，例如: a^2 + b^2 = c^2)", "核心公式2 (必须双斜杠转义，例如: \\\\sqrt{a} + \\\\sqrt{b})"],
    "keyPoints": ["关键要点1", "关键要点2"]
  },
  "questions": [
    {
      "type": "multiple-choice | short-answer | drawing",
      "question": "题目内容 (支持 LaTeX，例如求 $99 \\\\times 101$)",
      "options": ["A. 选项", "B. 选项", "C. 选项", "D. 选项"], 
      "hint": "提示信息 (可选)",
      "answer": "正确答案",
      "explanation": "详细解题过程 (支持 Markdown 与双反斜杠的 \\\\times 等)"
    }
    // 注意：这里的题目数量必须是 4 到 6 个！
  ]
}
```
